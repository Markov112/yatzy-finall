# Tekstipohjainen Yatzy-peli

## Projektin idea
perinteisen yatzyn tylinen noppa peli noppaaa heitetään ja pisteet mahdollista laskea

## Toiminnallisuus
- Pelaaja heittää viittä noppaa
- Nopat tallennetaan listaan
- Pelaaja voi:
  - heittää uudelleen
  - tarkastella pisteitä
  - lopettaa pelin
- Ohjelma laskee automaattisesti tulokset

## Toteutustapa
- Ohjelma on jaettu selkeisiin aliohjelmiin
- Käytetään listoja noppien tallentamiseen
- Silmukka pitää pelin käynnissä
- Koodi on kommentoitu
